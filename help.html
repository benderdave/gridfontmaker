GridfontMaker v1.1

Gridfontmaker is a tool for designing Gridfonts as described in Fluid Concepts and Creative Analogies, by Douglas Hofstadter. A gridfont is a font made up of 26 glyphs representing the lowercase letters of the English alphabet. Each letter is drawn on a 3x7 grid with connections allowed only between neighbors (including 45 degree diagonals.)

Briefly, the editor contains four areas, from top to bottom:

  • Font name
  • Small a-z display
  • Editable letters
  • Editable text area

      • Font name

      This area allows you to enter or change the current name of the font. We recommend you include the name of the designer, e.g. "Curiosity by Sandy Swan".

      Small a-z display

      This area contains a small rendering of the lowercase alphabet (a-z) in the current font. Use +/- to change the font size when this area is selected (click with the mouse to select.) You can also add other gridfonts to this area so that you can compare an existing font to the one you are currectly working on.

      Editable letters

      This area contains two rows with all 26 letters. Edit each letter in-place by dragging the mouse between anchor points to create a stroke, or by clicking on an empty spot where a new stroke can be added. Delete strokes by clicking on them.

      When hovering over a letter..

      • Use arrows to nudge the strokes up, down, left and to the right.
      • Press h to flip the letter horizontally.
      • Press v to flip the letter vertically.
      • Press r to rotate the letter 180 degrees around its center.
      • Press c to clear the letter.
      • Press a to clear all letters except this one. (It's common for a letter in one typeface to become the seed for a completely different, new typeface.)
      • Press o to see the order of strokes. Stroke numbers are displayed nearest to their starting anchor.

      When hovering over a stroke..

      • Press f to flip the direction of the stroke.

      When hovering over the dark rectange at the bottom of a letter the letter is selected. Here..

      • Drag the mouse to copy the letter onto another letter.
      • Shift-Drag with the mouse to move the letter in between two other letters. (This is so you can work on two letters side-by-side if desired. Use the Sort action in the grid menu to restore the original order.)

      Editable text area

      Type any text here that you would like to see displayed in your font. Use +/- to change the font size when this area is selected. Hold down a CONTROL or ALT key when pressing +/- for finer adjustment.
